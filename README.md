<p>Este é um projeto desenvolvido durante a <strong><a href="https://nextlevelweek.com/" rel="nofollow">Next Level Week</a></strong>, realizada pela <strong><a href="https://github.com/Rocketseat">@Rocketseat</a></strong> durante os dias 3 a 9 de Agosto de 2020.</p>
<p>A proposta do projeto é uma aplicação que possa ligar quem deseja aprender, com quer ensinar. É possível encontrar alunos para o que você leciona, ou encontrar o professor para aquela matéria que você sempre quis aprender mais! Sem que haja nenhuma limitação de área!!</p>
